## Requirement

- PHP version ^8.0.2 (Command line php when running composer install)




## Installation 

- clone project 
```bash
  git clone https://github.com/khfiii/blog-website.git
```

- composer install

```bash
composer install
```

- build asset

```bash
npm run dev 
```

- generate env key

```bash
php artisan key:generate
```


- migrate

```
php artisan migrate
```


- running project

```
php artisan serve
```
